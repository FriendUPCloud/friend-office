About Document App

OnlyOffice front-end software is developed and maintained by Friend Software Labs.

OnlyOffice Document is a front-end and a set of improvements to the OnlyOffice Document Server and document editor. It implements important integrations to Friend OS for file support, drag & drop as well as other User Interface adaptations.

The source code for the OnlyOffice front-end is available at:

  • https://github.com/FriendUPCloud/friend-office

The source code for OnlyOffice is available at:

  • https://github.com/ONLYOFFICE
