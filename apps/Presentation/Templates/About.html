About Presentation App

Friend Office is developed and maintained by Friend Software Labs.

Friend Office Presentation is a front-end and a set of improvements to the OnlyOffice Document Server and presentation editor. It implements important integrations to Friend OS for file support, drag & drop as well as other User Interface adaptations.

The source code is available at:

  • https://github.com/FriendUPCloud/friend-office
