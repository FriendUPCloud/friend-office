About Friend Mail

Friend Mail is developed and maintained by Friend Software Labs.

Friend Mail is a front-end and a set of improvements to the OnlyOffice Community Server, mail and calendar clients. It implements important integrations to Friend OS for file support, drag & drop as well as other User Interface adaptations.

The source code for Friend Office is available at:

  • https://github.com/FriendUPCloud/friend-office

The source code for OnlyOffice is available at:

  • https://github.com/ONLYOFFICE
