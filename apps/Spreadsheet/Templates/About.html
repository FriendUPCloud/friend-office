About Spreadsheet App

Friend Office is developed and maintained by Friend Software Labs.

Friend Office Spreadsheet is a front-end and a set of improvements to the OnlyOffice Document Server and spreadsheet editor. It implements important integrations to Friend OS for file support, drag & drop as well as other User Interface adaptations.

The source code for Friend Office is available at:

  • https://github.com/FriendUPCloud/friend-office

The source code for OnlyOffice is available at:

  • https://github.com/ONLYOFFICE
